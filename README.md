# Mood-Habit-Tracker
The Mood & Habit Tracker is a simple python program that helpsa user keep track of their daily mood and personal habits like sleep, study, exercise, and water intake.

# What this project can do
~Record your daily mood

~Track your sleep, study, exercise, and water intake

~View all the saved records

~Get a weekly summary report

~See:

-Your most common mood

-Your best day and worst day

-Average sleep, study, exercise, and water intake

-Get simple suggestions to improve your habits

# Concepts Used
The project uses:
~Lists

~Dictionaries

~Loops(for & while)

~Conditional Statements(if, elif, else)

~Functions

~File handling

~Basic calculations(sum, average, max, min)
