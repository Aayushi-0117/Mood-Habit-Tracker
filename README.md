# Mood-Habit-Tracker
The Mood & Habit Tracker is a simple python program that helpsa user keep track of their daily mood and personal habits like sleep, study, exercise, and water intake.

# What this project can do
#-Add daily mood and habits details.
#-View all saved records
#-Get a weekly summary report
#-Find:
#~Most common mood
#~Best day
#~Worst day
#~Average sleep, study, exercise and water intake
#-Data is saved in a text file
#-Gives simple suggestions for improvement

# Concepts Used
The project uses:
#-Lists
#-Dictionaries
#-Loops(for & while)
#-Conditional Statements(if, elif, else)
#-Functions
#-File handling
#-Basic calculations(sum, average, max, min)
